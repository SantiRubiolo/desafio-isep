En el repositorio se encuentran dos carpetas: la carpeta "desafio-isep" contiene los archivos de la landing page en html (html,css,js) lista para subirla a cualquier hosting o acceder de forma local en el navegador. En la carpeta "desafio-isep-wordpress" se encuenta la misma landing page adaptada para subirla a los temas de wordpress.

Pasos para subir el tema a WordPress:

1. Copiar todo el contenido de la carpeta "desafio-isep-wordpress" en la carpeta themes donde esta instalado wordpress.
2. Ingresar a WordPress.
3. Ingresar a Apariencias/Temas.
4. Activar el tema "desafio-isep".
5. Visitar el sitio.
